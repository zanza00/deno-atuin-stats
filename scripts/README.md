# Terminal Stats Scripts

## Get Top Commands

This script extracts the top 10 most used commands from the stats file and the Atuin SQLite database.

### Usage

1. Ensure you have [Deno](https://deno.land/) installed.
2. Run the script using the following command:

    ```sh
    deno run -A stats.ts
    ```

3. The script will output the top 10 most used commands from the specified stats file and the Atuin SQLite database.
